# Blooma · Mercado Melchor Ocampo

Aplicación Flask adaptada para Render gratuito con SQLite y almacenamiento multimedia temporal.

## Cuentas de demostración

- Cliente: `cliente@demo.local` / `demo123`
- Dueño: `dueno@demo.local` / `demo123`
- Empleado: `empleado@demo.local` / `demo123`
- Administrador: `admin@demo.local` / `demo123`

Los datos nuevos y archivos subidos pueden reiniciarse cuando Render gratuito vuelve a desplegar el servicio.
